# 参数
表面着色点的参数坐标。对于区域光源，它在平面映射中输出其UV坐标，在球形坐标中输出点光源坐标。

# 背面
设定为1.0时，表示从背面观察；设定为0.0时，表示从正面观察。

# 尖锐度 `仅适用于Cycles引擎`
每个网格顶点的曲率近似值。较亮的值表示凸角，较暗的值表示凹角。可用于制作含有污垢的贴图，或制作磨损效果。

# 各孤岛随机 `仅适用于Cycles引擎`
网格的每个相连的元素(孤岛)的随机值。可用于向由分离单位(如树叶、木板或多个样条线的曲线)组成的网格添加变化。

**孤岛的概念：**
	在一个物体中不相连的部分为一个孤岛
